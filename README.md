# IBM HR Analytics - Employee Attrition Prediction

## 📌 Project Overview
This project predicts employee attrition using Logistic Regression. The dataset contains HR-related employee details such as job role, income, overtime, and years at company.

## 📊 Exploratory Data Analysis
- Attrition Count Plot
- Correlation Heatmap

## 🤖 Machine Learning Model
- Algorithm: Logistic Regression
- Feature Scaling: StandardScaler
- Class Weight: Balanced

## 📈 Model Evaluation
- Accuracy Score
- Classification Report
- Confusion Matrix

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Seaborn
- Scikit-learn

---

This project demonstrates classification modeling for HR decision support systems.
